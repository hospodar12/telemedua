# WebAR-сканер для Safari/iPad: фото → YouTube

Цей мініпроєкт відкриває посилання:

https://www.youtube.com/watch?v=-QZqMT6W9ZQ

після того, як браузер Safari через камеру впізнає задане фото.

## Що всередині

- `index.html` — готова сторінка WebAR-сканера.
- `assets/target.jpg` — фото-мітка, яке треба показати камері.
- `README_UA.md` — ця інструкція.

## Як запустити на iPad

Варіант для швидкого тесту:

1. Завантажте папку на будь-який HTTPS-хостинг: GitHub Pages, Netlify, Vercel або сервер університету.
2. Відкрийте `index.html` у Safari на iPad.
3. Натисніть «Почати сканування».
4. Дозвольте доступ до камери.
5. Наведіть камеру на це саме фото.
6. Після розпізнавання відкриється YouTube.

Важливо: камера в Safari працює лише через HTTPS або localhost. Якщо просто відкрити HTML-файл із файлової системи, камера може не запуститися.

## Як вставити в підручник

Поруч із фото в підручнику розмістіть QR-код, який веде на адресу цієї WebAR-сторінки. Студент сканує QR-код, відкриває сторінку, дозволяє камеру і наводить iPad на фото.

## Налаштування

У `index.html` є рядок:

```js
const YOUTUBE_URL = "https://www.youtube.com/watch?v=-QZqMT6W9ZQ";
```

Тут можна замінити посилання на інше відео.

Також є параметр:

```js
const MIN_SCORE = 0.60;
```

Якщо відео відкривається надто рано — збільшіть до `0.68` або `0.75`.
Якщо фото погано розпізнається — зменшіть до `0.50` або `0.55`.

## Обмеження

Це демонстраційна реалізація без встановлення застосунку. Вона використовує браузерне розпізнавання зображення і найкраще працює, коли iPad наведено на фото майже прямо, без сильного нахилу, при доброму освітленні. Для промислової версії підручника краще використовувати MindAR / 8th Wall / Zappar або власний iOS-застосунок на ARKit.
